    2. 数组的优缺点
        优点: 
            构建非常简单;
            能在O(1)的时间里根据数组的下标index查询某个元素
        缺点: 
            构建时必须分配一段连续的空间
            查询某个元素是否存在时需要遍历整个数组, 耗费O(n)时间.n为元素的个数
            删除和添加某个元素时, 同样需要耗费O(n)的时间
